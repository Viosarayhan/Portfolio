Tentang Saya

Halo, nama saya Viosa Rayhan Pratama. Saya seorang mahasiswa yang saat ini sedang mengikuti program Studi Independen di Celerates, berfokus pada pengembangan Web dan desain UI/UX. Dalam program ini, saya mengasah kemampuan untuk merancang dan mengembangkan website yang interaktif, fungsional, dan ramah pengguna. Saya bersemangat dalam menciptakan solusi digital yang dapat meningkatkan pengalaman pengguna.

Saya memiliki pengalaman dalam menggunakan alat seperti Figma untuk membuat prototipe, serta pengetahuan mendalam tentang teknologi web seperti HTML, CSS, JavaScript, dan framework modern lainnya. Dengan komitmen untuk terus berkembang, saya siap untuk menghadirkan solusi inovatif yang dapat memberikan dampak positif bagi pengembangan teknologi.

Jika Anda tertarik untuk berkolaborasi atau memiliki pertanyaan, jangan ragu untuk menghubungi saya melalui kontak yang tersedia di bawah ini.
